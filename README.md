# Student Performance Prediction

A machine learning project that predicts student performance based on various academic and lifestyle factors. 

## Project Workflow
1. **Data Preprocessing:** Handled missing values, mapped ordinal data, encoded categorical variables (Label Encoding, One-Hot Encoding), and scaled numerical features using `Pandas` and `StandardScaler`.
2. **Machine Learning Model:** Trained a Linear Regression model using `scikit-learn` to predict the overall academic score.
3. **Evaluation:** Assessed the model's accuracy using R2 Score and Mean Absolute Error (MAE).

## Tech Stack
* Python
* Pandas
* Scikit-Learn
